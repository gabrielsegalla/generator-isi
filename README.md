# Generator ISI
Scaffolding a nextJS APP to consume Jhipster API

## Prerequisites

- [Node.js](https://nodejs.org/) installed
- [npm](https://www.npmjs.com/)

## Getting Started

1. Install this repository:

   ```bash
   npm install -g https://github.com/gabrielsegalla/generator-isi.git
    ```

2. Run:

   ```bash
   yo generator-isi
    ```
